Enumeration defining the typical types of boilers. The _IfcBoilerTypeEnum_ contains the following:

* **WATER**: Water boiler.
* **STEAM**: Steam boiler.
* **USERDEFINED**: User-defined Boiler type.
* **NOTDEFINED**: Undefined Boiler type.
