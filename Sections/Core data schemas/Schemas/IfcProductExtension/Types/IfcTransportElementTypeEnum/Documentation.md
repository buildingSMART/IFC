This enumeration is used to identify primary transport element types. The _IfcTransportElementTypeEnum_ contains the following:

*  **ELEVATOR**: Elevator. 
*  **ESCALATOR**: Escalator. 
*  **MOVINGWALKWAY**: Moving walkway.
